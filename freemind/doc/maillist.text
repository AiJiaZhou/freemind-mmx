<H2>Mailing Lists</H2>
We have the following mailing lists for FreeMind:<br>
<A HREF="http://lists.sourceforge.net/mailman/listinfo/freemind-announce"><H4>freemind-announce</H4></A>
A mail is sent to this list whenever a new Version of FreeMind is released. The traffic will be <b>very</b> low.<br>
<A HREF="http://lists.sourceforge.net/mailman/listinfo/freemind-users"><H4>freemind-users</H4></A>
This list is intended for <b>you</b>, the users of FreeMind. I will be on this list too and answer questions about FreeMind. It depends on you how much traffic will be on this list.<br>
