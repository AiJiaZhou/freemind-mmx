Installation

How can I install Freemind?

 FreeMind requires Java 1.4 or higher If Java is not installed, please abort the installation and download the JRE - Java Runtime Environment from here.

Usage

Customization

Using another browser

 If you want to use another browser, edit the user.properties file and change the following property according to your operation system:
  • default_browser_command_windows_nt
  • default_browser_command_windows_9x
  • default_browser_command_mac
  • default_browser_command_other_os

Editing the user properties

 The user.properties file resides in a subdirectory called "freemind" of your home directory. As an example, under Windows 2000, the file is normally in:
  • C:\Documents and Settings\<username>\freemind
