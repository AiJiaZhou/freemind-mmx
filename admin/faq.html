Installation

How can I install Freemind?

 FreeMind requires Java 1.4 or higher If Java is not installed, please abort the installation and download the JRE - Java Runtime Environment from here.

I installed Java JRE and FreeMind, execute freemind.exe (or the icon with the butterfly) and nothing happens. Why?

 Your Java installation is buggy. Try to excecute using "Run command..." in the start menu the command "javaw.exe". If this fails, you have to arrange your path settings, such that the java/bin directory is in your path.

Or try to start the batch "freemind.bat", read the messages and react.

Usage

How can I create a graphical link?

 This is done by Drag&Drop using the mouse, click on a node, hold the Shift and the Control keys and don't release them now, move this node to the destination node, release the mouse button (and now you may also release the keys, but not earlier), and there is your link.

Customization

Using another browser

 If you want to use another browser, edit the user.properties file and change the following property according to your operation system:
  • default_browser_command_windows_nt
  • default_browser_command_windows_9x
  • default_browser_command_mac
  • default_browser_command_other_os
You have to write something like
  • default_browser_command_windows_nt = C:\Program Files\Internet Explorer\iexplore.exe "{0}"

Editing the user properties

 The user.properties file resides in a subdirectory called "freemind" of your home directory. As an example, under Windows 2000, the file is normally in:
  • C:\Documents and Settings\<username>\freemind

Common Problems

Cannot open the Documentation in the Help menu

 The path doc/maps/freemind.mm is not found. This is due to your own user.properties file which contains the wrong path from the last version 0.6.7 of freemind. Yust delete the entry docmapurl = ./doc/freemind.mm from the user.properties.

If there are more than one link in the same area, I cannot change the properties of the link

 The reason is that right clicking these links, the first link is taken that intersects a rectangle around the cursor. To get the other link(s), you have to find a place without other interfering links or you have to fold other links, such that they do not disturb.
